# Curriculum

## Design principles

- **Each cohort owns its entire tool stack.** Nothing handed down from the program or a prior cohort.
- **GitHub-native everything.** Every artifact, review, and decision leaves a public trail.
- **Agent-first.** Students direct research, development, and QA agents from week 1.
- **Repeat enrollment is normal.** Single fee, unlimited cohorts; anyone wanting more reps re-enrolls.
