# Project 2: Internal comms platform (Week 3)

**Purpose:** Same [loop](../the-loop.md) as Project 1, one week. Deliverable: internal communications platform. Winner replaces Discord as the cohort's comms layer. **All work tracked on the live PM platform** — first project managed in cohort-owned tooling.

---

## Kickoff brief (Tuesday week 3, 10:00)

> **Problem:** You're coordinating on Discord — bootstrap tooling, explicitly temporary. Build the comms layer this cohort actually uses through pilot end.
>
> **Users:** 30 students + 2 staff. Async-first. High-traffic during review weeks and deadlines.
>
> **Must handle:** Channels (or equivalent), direct messages, announcements, notifications, message persistence, search.
>
> **Integration foreshadow:** Week 5 you'll unify with PM and showcase. Use the same user email/ID as the PM platform from day 1.
>
> **Migration:** Winner cutover by Mon after week 3 vote. Discord `#announcements`, `#reviews`, `#general` **decommissioned** within 48 hrs.
