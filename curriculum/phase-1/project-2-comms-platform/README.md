# Project 2: Internal comms platform (Weeks 5–6)

**Purpose:** Same [loop](../the-loop.md) as Project 1, compressed to 2 weeks. Deliverable: internal communications platform. Winner replaces Discord as the cohort's comms layer. **All work tracked on the live PM platform** — first project managed in cohort-owned tooling.

---

## Kickoff brief (Tuesday week 5, 10:00)

> **Problem:** You're coordinating on Discord — bootstrap tooling, explicitly temporary. Build the comms layer this cohort actually uses for the next 11 weeks.
>
> **Users:** 30 students + 2 staff. Async-first. High-traffic during review weeks and deadlines.
>
> **Must handle:** Channels (or equivalent), direct messages, announcements, notifications, message persistence, search.
>
> **Integration foreshadow:** Week 7–8 you'll unify with PM and showcase. Use the same user email/ID as the PM platform from day 1.
>
> **Migration:** Winner cutover by Mon Oct 20. Discord `#announcements`, `#reviews`, `#general` **decommissioned** within 48 hrs.

| Aspect | Project 1 | Project 2 |

| Project tracking | Discord + sheet | **Live PM platform** |
