# Comms platform: operator handbook

Same SLA framework as [PM operator handbook](../project-1-pm-platform/operator-handbook.md). Comms-specific additions below.

---

## Comms-specific SLAs

| Metric | Target |
|--------|--------|
| Message delivery | < 2 sec median |
| Uptime | ≥ 99% business hours |
| Moderation response | Harmful content removed ≤ 2 hrs |
| DM privacy | No operator reading DMs unless abuse report |

---

## Integration obligations (weeks 7–8)

| Integration | Minimum |
|-------------|---------|
| PM notifications | Post to `#dev` or equivalent when task assigned (webhook or manual MVP) |
| Showcase contact | `#hiring-partners` channel or equivalent for partner inquiries |
| Shared identity | Email matches PM platform ([ecosystem-unification.md](../ecosystem-unification.md)) |

---

## Moderation

Operator may:
- Remove messages violating code of conduct
- Mute users temporarily (≤ 24 hr) for spam
- Escalate to program director for removal

Operator may **not** read DMs except on reported abuse ticket with director approval.
