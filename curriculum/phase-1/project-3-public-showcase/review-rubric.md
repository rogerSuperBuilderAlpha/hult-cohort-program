# Public showcase: review rubric

Dimension 5 swapped for **outward-facing quality**.

## Rubric dimensions (1–5)

| # | Dimension | 5 = excellent |
|---|-----------|---------------|
| 1 | Production readiness | Fast, stable, mobile OK |
| 2 | Core functionality | All baseline features including PM integration |
| 3 | Code quality | Clean, secure (no leaked keys), maintainable |
| 4 | Ecosystem thinking | Profiles tell a hiring story; links work; opt-out respected |
| 5 | **Credibility to employers** *(replaces generic UX)* | A hiring partner would take 10 min here and schedule interviews |

### Dimension 5 detail

| Score | Criteria |
|-------|----------|
| 1 | Looks like student homework; broken links |
| 3 | Functional portfolio; partner could use it |
| 5 | Would share with hiring manager unprompted; PM integration feels real |

## External signal (advisory, not vote weight)

During review week, showcase deploy may receive real visitor traffic. **Analytics count as rubric evidence** in reviews ("47 unique visitors during review week") but do not replace vote.
