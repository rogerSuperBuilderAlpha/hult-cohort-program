# Admissions grading rubric (staff only)

15 minutes per PR.

| Criterion | Pass | Fail |
|-----------|------|------|
| PR opened on time | Yes | No PR |
| Tests pass | `npm test` green | Red |
| Bug 1 fixed | getTasks returns copy | — |
| Bug 2 fixed | updateTask merges | — |
| Bug 3 fixed | PATCH uses req.params.id | — |
| Feature | DELETE route + deleteTask (optional pass if 3 bugs fixed well) | — |
| PR template | Filled, agent section honest | Empty |
| Plagiarism | Original work | Identical to another applicant |

**Pass:** 3 bugs fixed + tests green + mergeable PR.
**Strong pass:** + DELETE feature + clear PR description.
