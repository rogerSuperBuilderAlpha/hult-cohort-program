## What it is

A 6-week GitHub-native developer pilot. ~30 students ship cohort-owned infrastructure (PM platform, comms stack, public showcase), unify the winning stack, and complete final-sprint market/OSS deliverables using agent-first workflows (Cursor + Claude Code). Each cohort owns its entire toolchain — nothing inherited from prior cohorts.

## Student profile

- Strong GitHub hygiene and PR discipline
- Comfortable with AI-assisted development (demonstrated in admissions PR)
- Phase 2 bar: ≥25 external users per app; ≥1 merged OSS PR to qualified repo
