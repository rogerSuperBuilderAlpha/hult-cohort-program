## What it is

A one-semester, GitHub-native developer cohort. ~30 students ship three cohort-owned products (PM platform, comms stack, public showcase) using agent-first workflows (Cursor + Claude Code). Each cohort owns its entire toolchain — nothing inherited from prior cohorts.

| Benefit | Detail |

| **Showcase access** | Week 16 demo day — meet teams with shipped apps and OSS contributions |
| **Referral placement** | 25% fee on qualifying hires (FT SWE ≥ $80k base); 90-day clawback |
| **Brand visibility** | Logo on cohort showcase site; optional lightning-demo sponsor slot |

## Student profile

- Strong GitHub hygiene and PR discipline
- Comfortable with AI-assisted development (not copy-paste — demonstrated in admissions PR)
- Phase 2 bar: ≥25 external users per app; ≥1 merged OSS PR to repo ≥1k stars
