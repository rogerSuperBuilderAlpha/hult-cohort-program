# Cohort project template

Copy this repo when starting any Phase 1 or Phase 2 build. Push to `hult-cohort-{term}-{campus}/{project}-{handle}`.

## Includes

- `AGENTS.md` — document your agent workflow per project
- `docs/agents/` — research, dev, QA starter prompts
- `.github/` — issue + PR templates ([github-workflow.md](../../curriculum/onboarding/github-workflow.md))
- `LICENSE` — MIT

## Quick start

1. Use this template → name repo `pm-yourhandle`, `comms-yourhandle`, etc.
2. Fill `AGENTS.md`
3. Deploy to Vercel; put URL in README
4. Ensure eligible build checklist before deadline ([the-loop.md](../../curriculum/phase-1/the-loop.md))
