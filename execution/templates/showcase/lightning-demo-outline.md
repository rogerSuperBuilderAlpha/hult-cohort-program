# Lightning demo — 90 seconds

One slide or one screen. Strict timer at showcase.

## Structure (15 sec each × 6 beats)

1. **Who I am** — name, GitHub handle on screen
2. **Phase 1** — best moment: won operator / best build / key PR to cohort platform
3. **Learning app** — topic + user count
4. **Venture** — problem + traction or investor touch
5. **Open source** — merged PR repo + link
6. **Ask** — role type I'm looking for

## Visual

- QR code → showcase profile
- One screenshot max per beat
- No live demo (links only)

## Do not

- Read bullet paragraphs
- Apologize for metrics
- Exceed 90 seconds (cut venture first)
