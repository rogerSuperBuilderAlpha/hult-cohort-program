## Student presentation

Each student presents their **eight-project arc in 90 sec**, led by strongest artifact:

1. Phase 1: best build or operator role
2. Learning app user count
3. Venture traction or investor touch
4. Open source merge (or best open PR)
5. GitHub profile QR on slide

Template: `showcase-template/LIGHTNING_DECK.pptx` (1 slide).
